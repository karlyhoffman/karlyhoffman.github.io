About

Hi, my name is Karly.
I'm a web developer based in Chicago.

As a developer with over eight years of experience, I specialize in crafting responsive, user-friendly web interfaces that balance performance and design.

Currently, as a developer at a design studio, I focus on building and maintaining websites using Craft CMS, a PHP framework, ensuring scalable, maintainable architectures are tailored to each project's needs. My expertise extends to creating custom plugins, implementing performant animations, and optimizing user interactions to deliver polished digital experiences.

With a strong foundation in frontend development, I bring a detail-oriented approach to translating static mockups into reusable components that adapt seamlessly across devices. Alongside a collaborative team, I contribute to code reviews, system improvements, and hosting configurations. My passion lies in enhancing user journeys through thoughtful UX animations, supported by a commitment to clean, organized code and best practices.I gained experience writing and organizing content for web pages as a web editor. Now I bring that content to life through web development.

With an eye for detail and the ability to add functionality to any page or application, I enjoy piecing together the logic to build creative and interactive websites.
