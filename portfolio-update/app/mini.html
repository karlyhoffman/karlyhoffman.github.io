Photo Carousel

Interactive Map

A map with 10+ markers placed in different locations. When a user clicks on a map marker, an info-window opens displaying a photo slider and copy based on the info-window's location. Info-windows also contain direction buttons where users can input a starting address to display directions to that specific map marker.

Modals

Buttons that display modal forms on click.
